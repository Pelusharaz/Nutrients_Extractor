## Next.js PDF Allergen & Nutrition Extractor



## Structure
- pages/
  - index.js
  - api/upload.js
- lib/
  - ocr.js
  - extractPrompt.js
- components/
  - ResultTable.jsx

Install dependencies:
```
npm install next react react-dom openai pdf-parse tesseract.js multer
```

Run:
```
npm run dev
```
